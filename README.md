# Criando-um-Dashboard-de-Vendas-do-Xbox-com-Excel
Criar um dashboard de vendas do desafio
Foi utilizado TOTAL SUBSCRIPTIONS EA PLAY SEASON PASS e TOTAL SUBSCRIPTIONS MINECRAFT SEASON PASS para mostrar total de vendas
Segundo gráfico mostra a total de vendas por mês de todos os tipos de subscrição

O filtro possui seleção de Annual, Monthly e Quartely
